## Assets and Resources Specification

---

## Interactive Tools (Skill Builders)

### 1. Inflation Calculator

**Purpose:** Calculate purchasing power erosion and real vs. nominal values

**Functionality:**
- Input: Amount, years, inflation rate
- Output: Real value, purchasing power loss
- Comparison mode: Multiple inflation scenarios
- Reverse calculation: What nominal amount needed to equal real value?

**Technical:** JavaScript calculator with compound interest formulas, Chart.js visualization

---

### 2. Salary Inflation Adjuster

**Purpose:** Compare job offers and calculate real salary growth

**Functionality:**
- Input: Starting salary, annual raise %, years, inflation rate
- Output: Real salary trajectory, comparison charts
- Multiple offers comparison
- Break-even analysis

---

### 3. Investment Return Calculator (Inflation-Adjusted)

**Purpose:** Calculate real returns on investments after inflation

**Functionality:**
- Asset allocation tool ($10,000 across options)
- Real return calculations for stocks, bonds, TIPS, savings, real estate
- Risk/return visualization
- Diversification recommendations by age

---

### 4. Unemployment Protection Scorecard

**Purpose:** Assess personal readiness for unemployment

**Functionality:**
- Emergency fund calculator (months of expenses)
- Skills inventory and gap analysis
- Network strength assessment
- Financial flexibility score
- Personalized recommendations

## Implementation Notes

**For Developers:**
- All calculators must handle edge cases (negative inflation, extreme values)
- Real-time data integration from BLS API (optional, can use static)
- Mobile-responsive design essential
- Save user inputs for comparison

**For Content Team:**
- Update price comparisons annually
- Verify current inflation and unemployment rates
- Refresh investment return assumptions
- Update {{STATE_NAME}}-specific data

**For Teachers:**
- Calculators are core to learning - ensure all students have access
- Real vs. nominal concept is challenging - emphasize repeatedly
- Investment allocation activity very engaging - allow sufficient time
- Wage negotiation role-play effective if time permits
